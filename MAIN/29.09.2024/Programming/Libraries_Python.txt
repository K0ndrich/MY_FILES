
-----   Библиотеки Для Основы Python   ----------------------------------------------------------------------------------------------------------------------------------------------------
- ipython                  -> улучшение для интепритатора python, вызваеться в терминале как ipython
- time                     -> можна замедлять выполнение кода 


-----   Библиотеки Для Django   ----------------------------------------------------------------------------------------------------------------------------------------------------
- django                   -> ето фремворк django
- psycopg2                 -> для подключения базы данных PostgreSql к проекту django
- coverage                 -> для проверки написали ли мы тесты в django коде
- django-environ           -> для работы с переменными окружения
- gunicorn                 -> веб-сервер для запуска django проекта 

-----   Библиотеки Для Django_REST   ----------------------------------------------------------------------------------------------------------------------------------------------------
- django-filter            -> добалвяет фильтрацию для django_rest
- social-auth-app-django   -> добавляет авторизацию пользователей через социальныне сети
